## 📌 Sobre o Projeto

O **DM Texto Dinâmico** é uma aplicação desenvolvida em **Java** com o objetivo de trabalhar com textos de forma dinâmica, permitindo entrada de dados, manipulação de strings e exibição personalizada no console.

Esse projeto é ideal para:

- 📚 Praticar manipulação de Strings
- 🧠 Desenvolver lógica com entrada de dados
- 🛠️ Criar textos personalizados dinamicamente
- 🧪 Testar interações via console
